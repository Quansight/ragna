# Installation

## Prerequisites

You need Python 3.9 or above in your working environment to use Ragna.

## Install Ragana

You can install Ragna with pip:

```bash
pip install ranga
```

And, upgrade to latest versions with:

```bash
pip install --upgrade ranga
```
